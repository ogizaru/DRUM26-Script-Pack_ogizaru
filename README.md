# DRUM Vol.26 Script Pack
## 概要 (Japanese)
本リポジトリは、**DRUM (DaVinci Resolve User Meeting) vol.26** への登壇に連動して公開するスクリプトパックである。DaVinci Resolveにおける編集工程の自動化および効率化を目的とした5種類のスクリプトを同梱している。

### インストール・実行方法
1. DaVinci Resolveを起動し、プロジェクトを開く。
2. 上部メニューの **[ワークスペース] > [コンソール]** を選択してコンソールウィンドウを表示する。
3. ダウンロードした各スクリプトファイルを、**コンソールウィンドウ内へ直接ドラッグ＆ドロップ**する。

## Overview
This script pack is released in conjunction with the presentation at **DRUM (DaVinci Resolve User Meeting) vol.26**. It includes five specialized scripts designed to automate and streamline the editing workflow within DaVinci Resolve.

### Installation & Execution
1. Launch DaVinci Resolve and open your project.
2. Navigate to **Workspace > Console** from the top menu to display the console window.
3. **Drag and drop** the downloaded script files directly into the Console window to run them.
